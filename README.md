# 🛡️ SteamGuard iOS Client (Unofficial)

![Platform](https://img.shields.io/badge/Platform-iOS-blue.svg)
![Swift](https://img.shields.io/badge/Language-SwiftUI-orange.svg)
![Status](https://img.shields.io/badge/Status-Beta-yellow.svg)
![Build](https://github.com/OutTuna/SDA-IOS/actions/workflows/swift.yml/badge.svg)

**Нативный iOS клиент для файлов Steam Desktop Authenticator (SDA).**
Приложение позволяет использовать файлы `.maFile` на iPhone для генерации кодов двухфакторной аутентификации (2FA), управления трейдами и подтверждениями торговой площадки без необходимости перепривязывать аутентификатор к официальному приложению Steam.

---

## ✨ Основные возможности (Features)

* 📂 **Импорт аккаунтов:** Полная поддержка импорта `.maFile` (экспортированных из SDA) ***включая зашифрованные файлы (с вводом пароля)***. (дешифрование в бете!!!!)
* 🔐 **Биометрическая защита:** Вход в приложение и доступ к кодам надежно защищены с помощью **Face ID / Touch ID** (или пасскода).
* 🔑 **Генерация кодов:** Точная генерация 5-значных Steam Guard кодов (TOTP) в реальном времени с удобным копированием по тапу.
* 🤝 **Подтверждения (Trade & Market):** Просмотр, подтверждение и отмена обменов, продаж на торговой площадке и смены номера телефона. Поддерживается функция массового принятия ("Принять все").
* 🌐 **Встроенная авторизация:** Удобный вход в аккаунт Steam через встроенный WebView (для автоматического получения `steamLoginSecure` cookie).
* 🎨 **Native UI:** Чистый интерфейс на **SwiftUI** с полной поддержкой темной темы.
* 🔄 **Мультиаккаунт:** Удобное переключение между неограниченным количеством аккаунтов.

## 🛠 Установка и Запуск

1.  Клонируйте репозиторий или скачайте исходный код.
2.  Откройте файл `.xcodeproj` в **Xcode**.
3.  Скомпилируйте и запустите на симуляторе или реальном устройстве (требуется iOS 17.0+).

### Как добавить аккаунт:
1.  Откройте **SDA** на компьютере.
2.  Сделайте экспорт файла:
    * **Для незашифрованных аккаунтов:** `File` -> `Export Account` -> `Export to plain text`.
    * **Для зашифрованных аккаунтов:** Потребуется сам `.maFile` и файл `manifest.json`.
3.  Передайте файлы на iPhone (через AirDrop или iCloud Drive / приложение "Файлы").
4.  В приложении нажмите **"+"** в правом верхнем углу.
5.  Выберите нужный `.maFile` (если файл зашифрован, необходимо выделить **одновременно** `.maFile` и `manifest.json`, после чего приложение запросит пароль расшифровки).
